# Python – Introduction

This section marks the beginning of the **LWS – Programming Fundamentals 1001** series.

The goal of this module is to introduce:
- What programming is
- What Python is and why it is used
- How Python code works internally
- How to write and run your first Python program

This section is intentionally **concept-focused** and does not assume any prior programming experience.

---

## Files in This Section

- `introduction.py`  
  Introduces programming and Python at a conceptual level.

- `how_python_works.py`  
  Explains how Python code is executed internally.

- `first_program.py`  
  Walks through writing and understanding the first Python program.

---

## Learning Outcome

After completing this section, learners will:
- Understand the purpose of programming
- Know why Python is a good starting language
- Understand how Python executes code
- Be comfortable running a basic Python program
